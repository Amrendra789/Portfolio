E-commerce - Platform for Selling of farmed products

  • Tech Stack: HTML, CSS, JavaScript, React, MySQL, Mongo DB, Node and Express.
  • Github URL: Project Link

Description: Created a dynamic e-commerce platform addressing the unique challenges faced by both buyers and sellers in the agricultural sector. This platform streamlines the process of sourcing and distributing farm products, revolutionizing the way farmers connect with consumers.

Key Achievements: Empowered economically disadvantaged farmers by providing free training and support through the MGNREGA program, enabling them to optimize their yields and income. Successfully integrated essential features including user authentication, user-focused content, robust contact information and support, and seamless payment processing to ensure a smooth and secure transaction experience.

Technologies Utilized: Implemented a front-end using EJS, CSS, and JavaScript to create an intuitive and user-friendly interface. Leveraged Node.js and Express.js to develop a robust and scalable back-end infrastructure, ensuring seamless functionality and responsiveness. Employed MongoDB for efficient data management, enabling swift retrieval and storage of critical information. This e-commerce platform not only addresses the specific needs of farmers but also provides a reliable and accessible marketplace for agricultural products, contributing to the growth and sustainability of the farming community.
